创建之前，请先确认安装了node.js，和vue的脚手架和webpack

1、创建

  1. 如果要在D盘新建项目，在cmd命令窗口转到D盘，运行

     Vue init webpack vuedemo

  2. 之后会有一系列提示，直接默认就可以。最后执行结束后

     cd vuedemo
 npm run dev

  3. 访问http://localhost:8080即可预览。

  4. 创建后文件目录讲解,更具体的自行百度。

上面是基本架构目录，结合我们现在的目录再说一下：

  1. build/utils.js 当初是springboot项目，将VUE打包后作为静态资源放在static目录下用的，前后端分离的需要测试下。
    publicPath: '../../' 

     if (options.extract) {
         return ExtractTextPlugin.extract({
         use: loaders,
         fallback: 'vue-style-loader',
        publicPath: '../../'     //解决部署后css路径不对问题 
         })        }

  2. onfig/index.js修改

     assetsPublicPath: '/',//解决静态文件路径问题
 autoOpenBrowser: true,//dev时，编译完成后自动打开浏览器
 useEslint: false,//取消语法检查

2、增加标签页icon

  1. favicon.ico文件放在根目录下

  2. webpack.prod.conf.js， webpack.prod.dev.js修改，在 inject: true 这一行下面增加

     favicon: path.resolve('./favicon.ico'),

  3. 在index.html下面新增

     <link rel="shortcut icon" type="image/x-icon" href="./favicon.ico" rel="external nofollow" />

3、深入

我们需要安装各种插件

    cnpm install element-ui –save（必须，如果不行换   cnpm i element-ui -S）
    cnpm install axios --save（必须）
    cnpm install vuex --save（必须）
    cnpm install js-md5 --save
    cnpm install vue-router --save（必须）
    cnpm install echarts --save
